GitHub PR Group — Options

Connexion GitHub

Pas encore de token ? Créer un token sur GitHub →

Scopes requis :
• public_repo — pour les repos publics uniquement
• repo — pour inclure les repos privés
• read:user — pour vérifier la connexion

Filtres

Un repo par ligne, au format owner/repo. Les PRs de ces repos n'apparaîtront pas dans le groupe.

PRs ignorées manuellement

PRs définitivement masquées : —

Quand vous fermez un onglet du groupe manuellement, la PR est définitivement ignorée. Cliquez "Tout réafficher" pour remettre à zéro.
